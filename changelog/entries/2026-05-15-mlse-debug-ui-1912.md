# 新增 mlse-debug 浏览器调试页面

## 摘要

本轮按用户要求实现 `cmd/mlse-debug`，目标是在浏览器中展示 Go 源码与 MLSE formal 指令的左右布局，便于调试前端输出。

## 变更

- 新增 `cmd/mlse-debug` 命令，支持 `mlse-debug [flags] <input.go>` 启动本地 HTTP debug 页面，并提供 `-addr` 与 `-open` 参数。
- 新增 `internal/debugview`，复用 Go formal frontend 输出，提取源码行、formal MLIR 指令与 `loc(...)` 映射关系。
- 新增浏览器页面：左侧源码、右侧指令，支持点击源码行筛选相关指令、搜索、按指令 kind 过滤，并提供 raw MLIR 入口。
- 更新 `scripts/build.sh` 和 `scripts/test-all.sh`，把 `mlse-debug` 纳入统一构建产物校验。
- 更新 `README.md`、`docs/dev-setup.md`、`docs/spec.md`，记录新的命令入口、使用方式和工具边界。
